Hướng dẫn publish với github và heroku

Chuẩn bị Nếu chưa có tài khoản Github vào trang https://github.com/ đăng ký tài khoản. Lưu ý cần xác nhận tài khoản trong email nhận được từ github. Đăng ký tài khoản trên Heroku https://www.heroku.com/ Máy tính cần cài đặt Git. Để kiểm tra mở CMD ( với windows) hoặc Terminal (với Mac Os)
